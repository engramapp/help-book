# Frequently Asked Questions

<details>
<summary>What operating systems does Engram Sync support?</summary>

Engram Sync supports every platform that Engram can run on. Currently that means Windows, macOS, Linux, Android and iOS.
</details>

<details>
<summary>What type of data is synced?</summary>

Only the data within your collection database and attached media are synced to Engram Servers.
</details>

<details>
<summary>How much storage do I have?</summary>
Please see [Storage limits](/sync/storage-limits)
</details>

<details>
<summary>How large can each media file be?</summary>
The current limit on media attachments is 100 MB.
</details>

<details>
<summary>Is my data being synced in the background?</summary>
No, files are only synced when Engram is running.
</details>

<details>
<summary>Do you keep my data if I refund my service?</summary>
No. The data is deleted immediately from Engram Sync servers. Data in your local collection is not affected.
</details>
